## About

ProgramIQ is a simulated enterprise program management portfolio dashboard built to demonstrate Senior TPM skills across SAFe, PMI, and Agile methodologies.

The dashboard tells the story of **Nova Systems**, a fictional Series C AI/SaaS company. The scenario centers on a blocked ML Platform deployment — gated on a SOC 2 compliance audit — that cascades into three downstream programs and puts $3.2M in Q2 ARR at risk. Every panel in the dashboard is evidence for that story.

## What It Demonstrates

- **SAFe methodology** — PI cadence, predictability metrics, ART dependency management, velocity tracking
- **PMI methodology** — risk register with severity banding, stakeholder sign-off chains, budget variance tracking
- **Executive communication** — status → risk → decision → milestones structure, quantified trade-offs
- **Dependency thinking** — visual network graph with critical path highlighting
- **Servant leadership** — escalation-first, transparency-first program management philosophy

## Key Features

| Feature | Description |
|---|---|
| **Executive Briefing** | One-page status → risk → decision → milestones view for leadership meetings |
| **Trade-Off Panel** | Decision scenarios with quantified revenue, schedule, cost, and risk impact |
| **Dependency Graph** | SVG network visualization with critical path highlighted |
| **Budget Tracking** | Spend vs. plan, forecast at completion, per-program variance |
| **Predictability Formula** | SAFe PI predictability with formula breakdown and confidence intervals |
| **Stakeholder Alignment** | Sign-off matrix across programs with comment visibility |
| **Forecast Timeline** | Completion forecasts with P10/P90 confidence bands |
| **My Approach** | Methodology page explaining servant leadership, escalation, and SAFe + PMI thinking |

## Tech Stack

React · TypeScript · Vite · Tailwind CSS · Recharts · shadcn/ui
